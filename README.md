# game-arrow-again

`Arrow Again / 箭了又箭` 独立 app 项目仓库。

## 当前原则

- 代码和文档统一维护在这个仓库里
- 默认目标平台：`Android + iOS`
- 默认路线：`非 Unity 优先`
- 当前玩法定义：箭头沿朝向飞出，路径无遮挡即可消除，错误点击扣生命值

## 当前目录

- `src/`：Web 原型与共享规则核验证
- `docs/`：项目记录、玩法分析、移动端架构
- `android/`：Android 原生壳占位
- `ios/`：iOS 原生壳占位

## 当前状态

- 已完成第一版 Arrow Again 规则核
- 已完成引导关 Web prototype
- 已完成 Android / iOS 双端启动架构文档
